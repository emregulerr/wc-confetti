== Description ==

You can "start a confetti rain and display a message" according to WooCommerce cart amount with Confetti for WooCommerce.

== Installation ==

1. Go to Plugins > Add New on the Dashboard menu
2. Search for the Confetti for WooCommerce and click the “Install” button of plugin
3. After installing the plugin click the “Activate” link.
